# Portscankun  
目標のポートが開いているか閉じているかを判断します。  
### OPTION   
`--tcp`  
ソケット通信をTCPで展開し、connect_exメソッドを使って直接的に通信を行いポートをスキャンします。ローカルにスキャンするならこっちを選んでください。  
`--syn`  
Scapyを使用し、SYNパケットをターゲットに送信し実際に通信しているようにみせかけた(3-way-handshakeを完全に行わない)スキャン方法です。ハーフスキャンと呼ばれるものです。  
`--ip IP`  
IP&ホスト名を指定。  
`--port PORT`  
スキャンする範囲を指定。
